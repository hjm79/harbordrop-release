## 🚀 Getting Started

### 1. Install the App

1. Download `HarborDrop.dmg` and drag the app to your **Applications** folder.
2. Launch HarborDrop — follow the onboarding guide on first launch.

### 2. Install the Browser Extension

Install the extension for your browser (see Download section above).

### 3. Verify Connection

1. Make sure HarborDrop is running.
2. Click the extension icon in your browser toolbar.
3. The popup should show **Connected** at the top.

## 📖 How to Download

### From the Extension Popup

Click the extension icon → browse detected media → click **Download**.

### Automatic Interception (Full Version)

With the Full Version extension, browser downloads are automatically handled by HarborDrop. Hold `Delete` key while clicking to bypass and use the browser's default downloader.

### Direct URL

Click the **+** button in the app toolbar and paste a URL.

### Drag & Drop

Drag any URL or link directly into the HarborDrop window.

### Right-Click Menu

Right-click any link on a web page → **Download with HarborDrop**.

### Video Overlay Button

A download button appears automatically over videos on web pages.

## ❓ FAQ

**Extension shows "Disconnected"**
1. Make sure HarborDrop is running.
2. Go to Settings → General → **Reinstall NMH**.
3. Reload the extension in your browser.

**Download failed**
- The video URL may have expired — refresh the page and try again.
- Use **Redownload from Source URL** to re-detect the media.

**"Extension context invalidated" error**
- Reload the extension, then reopen the tab.
